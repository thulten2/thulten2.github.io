HTML Terms

<!DOCTYPE>
Specifies which version of HTML the document will use
<head>
This tag holds elements in the HTML file, like a title
<title>
This tag is used to define the HTML document
<body>
This tag has all the info that will display on the screen
<footer>
This tag is where all the info at the bottom of the page goes
<p>
This tag will change the text to a paragraph format
<h1> - <h6>
This tag changes the text to a header format. There are 6 options
<b>
This tag will make the text bold
<i>
This tag will make the text have italics
<u>
This tag will add a underline to the text
<hr>
This tag will add a line across the screen
<ul>
This will create an unordered list
<ol>
This will create an ordered list
<li>
This will create a list item and where the data goes
<img>
This tag lets you insert images into the HTML document

CSS Terms

Style Rule
The selector, curly braces, and the diffrent lines with properties and values
Declaration Block
This is where the properties and their values appear
Declaration
Any single line of CSS that appears between the curly braces
Property
What appears before the colon in any line of CSS
Value
What appears right after the colon
Selector
Selects the most specific element to target.(#idName or .className)
Element type Selector
Selecting an element by its tag name instead of an id or class name
Pseudo-Class
Similar to a CSS class but it is not explicitly declared in the HTML
Combinator
Combining two selectors together that are related to eachother.Adding a space will refer to all the decendant selectors. Child with the > symbol and + for sibling selectors
